# SP-CSS

**SP-CSS** — a lightweight, responsive CSS utility + component library for modern web apps.  
Built and maintained by **Sphereka** — https://sphereka.com

> Open source. Released under the MIT License. Community contributions are welcome, but changes to protected branches require review and approval.

---

## Quick overview

- Minimal, pragmatic CSS with sensible defaults.
- Utilities-first approach with ready-made components: grid, buttons, forms, navs, cards, modals.
- Fully responsive out of the box.

---

## CDN / Installation

HTML
```
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Sphereka/SP-CSS@main/sp_css.css">
<script src="https://cdn.jsdelivr.net/gh/Sphereka/SP-CSS@main/sp.js"></script>
```
Manual
Include /sp_css.css and /sp.js from this repository in your project.
